## 프로젝트명

UACV (ULTRA AUTONOMOUS COMBAT VEHICLE)
라즈베리파이, 젯슨오린나노 Sensing Data를 활용한 자율 전투차 제작

## 기간 / 인원

2024.07.08 ~ 2024.08.16 (6주),  6명

## 개       요

병력 자원 감소와 인명 피해 감소를 위한 무인 자율 전투 차량 제작

## 기술 / 환경

#### 기술1 - 자율주행
LIDAR SLAM 기반 MAPPING 자율 주행

#### 기술2 - 소리 분석
MFCC 기반 모델로 총기 소리를 판단

#### 기술3 - 사격
표적 조준 및 포격
